# license-plate

## Goal
Calculate how many license plates you can generate using a specific format

## Description
I've always wanted to know the number of possibilities you get with the license plates you see in different countries.

Here are a few examples of license plates formats
* France: AA-123-AA
* Belgium: 1-AYR-538
* Netherlands: G-001-BB
* Spain: 2417 GSM

## Usage
### Build

`make`

Takes 1 argument with maximum 2 space characters.

### Execute

Examples:
1. France: `./license AA-123-AA`
2. Spain `./license 2417 GSM`
